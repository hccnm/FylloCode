通过 OpenAPI 查询组织列表。

## **请求语法**

```
GET https://{domain}/oapi/v1/platform/organizations
```

## **请求头**

| **参数**        | **类型** | **是否必填** | **描述**       | **示例值**                                      |
| --------------- | -------- | ------------ | -------------- | ----------------------------------------------- |
| x-yunxiao-token | string   | 是           | 个人访问令牌。 | pt-0fh3\\_\\_\\*\\*0fbG\\\_35af\\_\\_\\*\\*0484 |

## **请求参数**

| **参数** | **类型** | **位置** | **是否必填** | **描述**                                                                                                                      | **示例值** |
| -------- | -------- | -------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------- | ---------- |
| userId   | string   | query    | 否           | 用户 ID，标准版：返回个人访问令牌对应用户加入的组织列表，此参数无效；专属版：为空返回所有组织列表，不为空返回加入的组织列表。 |            |

## **请求示例**

```
curl -X 'GET' \
  'https://{domain}/oapi/v1/platform/organizations?userId={userId}&page={page}&perPage={perPage}' \
  -H 'Content-Type: application/json' \
  -H 'x-yunxiao-token: pt-0fh3****0fbG_35af****0484'
```

| name        | string   | 组织名称。   | 示例名                   |
